À propos

Funkywork est un espace de publication communautaire dont la ligne éditoriale est orientée vers la programmation et la création de Jeux-vidéo. Cependant, les auteurs sont assez libre des thématiques de leurs rédactions et il n'est pas à proscrire que certains s'essayent à des articles traitant de sujet (potentiellement) éloignés de cet axe principal.

Bien que l'objectif premier de cette page est d'avoir un outil adapté pour la publication d'articles, nous nous servirons aussi de cet espace pour diffuser du code (qu'il s'agisse de "logiciels", de "patchs" pour le logiciel RPG Maker ou autres diversités).

Le site web est maintenu via une application Erlang, les sources sont libres et disponibles via GitHub. Il est possible de contacter le "webmaster" via cette adresse: xavier.vdw@gmail.com.
